# VPN Configuration Guide

## Windows Setup
1. Download the VPN client from the IT portal
2. Install the application with administrator privileges
3. Use your domain credentials to connect
4. Select the appropriate server location based on your region

## macOS Setup
1. Open System Preferences > Network
2. Click the + button to add a new connection
3. Choose VPN from the Interface dropdown
4. Select VPN Type: IKEv2, L2TP, or PPTP as specified by IT
5. Enter the server details provided by IT
6. Configure authentication settings

## Connection Process
1. Launch VPN client application
2. Enter your username and password
3. Select server location if multiple options available
4. Click Connect and wait for established connection
5. Verify connection by checking your IP address

## Troubleshooting
- **Connection Fails**: Check your internet connection first
- **Authentication Error**: Verify your credentials are correct
- **Slow Performance**: Try different server locations
- **Can't Access Company Resources**: Ensure you're connected to correct VPN profile
- **Frequent Disconnects**: Contact IT if connection is unstable

## Security Best Practices
- Always connect to VPN before accessing company resources
- Use VPN when working from public WiFi networks
- Disconnect VPN when not needed to improve performance
- Report any connection issues to IT immediately
- Never share VPN credentials with others

For additional support or VPN client downloads, contact IT helpdesk.
